# foobar2000 无障碍增强组件更新日志

## 0.2.2

- 修复 `文件 -> 参数选项 -> 工具 -> foobar2000 无障碍增强 -> 关于` 页面中，焦点位于“主要用途”只读编辑框时按 `Shift+Tab` 会在页面控件内循环的问题。


## 0.2.1

- 修复组件自带 Tolk 运行库可能与其他 foobar2000 组件携带的同名 DLL 互相占用或复用的问题。
- 打包时将组件自带的 `Tolk.dll` 改名为 `TolkODS.dll`，并同步修改 Tolk 内部引用，确保组件优先加载自己的 Tolk 运行库。
- 将 Tolk 引用的 NVDA、保益、争渡相关 DLL 改为组件专用文件名，降低与朗读歌词组件或其他组件发生 DLL 冲突的风险。
- 新生成的安装包不再包含原始 `Tolk.dll`、`nvdaControllerClient32.dll`、`nvdaControllerClient64.dll`、`byctrl*.dll`、`ZDSRAPI*.dll` 文件名。
- 调整发布安装包命名和结构，改为仅提供 `x64` 与 `x86` 两个单架构安装包，格式为 `foo_output_device_switcher-版本号-架构.fb2k-component`。

## 0.2.0

- 组件显示名称统一改为 `foobar2000 无障碍增强`，DLL 文件名仍保持 `foo_output_device_switcher.dll`，方便后续升级维护。
- 更换 Tolk 运行库，来源为 <https://github.com/boz700908/tolk>。
- 新增 `播放 -> 播放模式` 子菜单，可查看当前播放模式，或切换到上一个/下一个 foobar2000 播放模式，并通过 Tolk 播报模式名称。
- 新增 `播放 -> 播放列表` 子菜单，可查看当前播放列表，或切换到上一个/下一个播放列表并播放目标列表中的焦点歌曲或第一首歌曲。
- 新增 `文件 -> 参数选项 -> 工具 -> foobar2000 无障碍增强` 参数页，接管“启动后隐藏 foobar2000 主窗口”功能，用于减少托盘模式下读屏读到幽灵窗口的问题。
- 参数页包括 `foobar2000 无障碍增强` 可展开分组，下面包含 `常规设置`、`TTS 语音` 和 `关于` 三个子页面。
- 新增 `TTS 语音` 参数页，支持在屏幕阅读器/Tolk 与 SAPI5.1 之间切换播报输出，并支持选择 SAPI5.1 语音、调整语速和试听。
- 新增 `关于` 参数页，包含组件主要用途说明、GitHub 仓库、GitHub 发布页和备用 gitee 发布页按钮。
- 迁移朗读歌词组件原有的启动隐藏主窗口配置，理论上已勾选过的用户安装本组件后会继续沿用原来的设置值。
- 加固播放模式切换的异常处理，避免读取或设置播放模式失败时影响 foobar2000 稳定性。
