# foobar2000 无障碍增强组件使用方法

`foobar2000 无障碍增强` 是一个 foobar2000 组件，用于在播放菜单中查看和切换 foobar2000 使用的播放设备，也可以切换 foobar2000 的播放模式、播放列表，并通过 Tolk 播报当前设备、切换后的设备名称、切换后的播放模式或播放列表名称。

## 适用环境

- Windows
- foobar2000 2.x，支持 x64 和 x86 版本
- foobar2000 1.5 / 1.6，使用 x86 兼容包

## 安装

foobar2000 2.x 可使用：

```text
foo_output_device_switcher.fb2k-component
foo_output_device_switcher-x64.fb2k-component
```

foobar2000 1.5 / 1.6 使用：

```text
foo_output_device_switcher-fb2k-x86.fb2k-component
```

foobar2000 1.x 是 32 位程序，请安装带 `fb2k-x86` 字样的安装包。后续 x86 包默认用于兼容 foobar2000 1.5 / 1.6。

在 foobar2000 中打开：

```text
文件 -> 参数选项 -> 组件
```

选择组件包安装，安装完成后重启 foobar2000。

## 菜单位置

安装后，菜单位置为：

```text
播放 -> 播放设备
播放 -> 播放模式
播放 -> 播放列表
```

`播放设备` 子菜单中包含：

```text
查看当前播放设备
切换到上一个播放设备
切换到下一个播放设备
```

`播放模式` 子菜单中包含：

```text
查看当前播放模式
切换到上一个播放模式
切换到下一个播放模式
```

`播放列表` 子菜单中包含：

```text
查看当前播放列表
切换到上一个播放列表并播放
切换到下一个播放列表并播放
```

## 使用方法

选择 `查看当前播放设备`，组件会播报 foobar2000 当前使用的播放设备。

播报格式为：

```text
当前 : 设备名称
```

选择 `切换到下一个播放设备`，foobar2000 会切换到设备列表中的下一个播放设备。

选择 `切换到上一个播放设备`，foobar2000 会切换到设备列表中的上一个播放设备。

设备列表会循环切换。例如当前已经是最后一个设备时，继续切换下一个会回到第一个设备。

选择 `查看当前播放模式`，组件会播报 foobar2000 当前使用的播放模式。

选择 `切换到下一个播放模式`，foobar2000 会切换到当前播放模式列表中的下一个模式。

选择 `切换到上一个播放模式`，foobar2000 会切换到当前播放模式列表中的上一个模式。

播放模式列表来自 foobar2000 自身，例如默认、重复、随机、单曲循环等；如果 foobar2000 或其他组件提供了不同的播放模式，菜单会按 foobar2000 当前返回的列表循环切换。

选择 `查看当前播放列表`，组件会播报 foobar2000 当前活动播放列表。

选择 `切换到下一个播放列表并播放`，foobar2000 会切换到播放列表列表中的下一个播放列表，并播放该播放列表中的焦点歌曲；如果没有焦点歌曲，则播放第一首歌曲。

选择 `切换到上一个播放列表并播放`，foobar2000 会切换到播放列表列表中的上一个播放列表，并播放该播放列表中的焦点歌曲；如果没有焦点歌曲，则播放第一首歌曲。

播放列表会循环切换。例如当前已经是最后一个播放列表时，继续切换下一个会回到第一个播放列表。

## 播报行为

切换成功后，组件会直接播报切换后的设备名称。

例如切换到耳机设备时，播报内容类似：

```text
扬声器 (Realtek(R) Audio)
```

查看当前播放设备时，播报内容类似：

```text
当前 : 扬声器 (MUV HW 1/2)
```

切换播放模式时，播报内容类似：

```text
播放模式 : 随机
```

查看当前播放模式时，播报内容类似：

```text
当前 : 随机
```

切换播放列表并播放时，播报内容类似：

```text
播放列表 : 默认
```

查看当前播放列表时，播报内容类似：

```text
当前 : 默认
```

## Tolk 文件

组件包中已经包含所需的 Tolk 文件，来源为：

```text
third_party\tolk-with-zdsr
```

安装组件包后不需要手动复制 `Tolk.dll`。

## 注意事项

- 本组件切换的是 foobar2000 的播放输出设备，不会修改 Windows 系统默认声卡。
- 如果 foobar2000 当前正在播放，切换设备可能会触发 foobar2000 重启输出流。
- 如果只有一个可用播放设备，菜单命令会显示为不可用。
- 如果只有一个可用播放模式，播放模式切换命令会显示为不可用。
- 如果只有一个播放列表，播放列表切换并播放命令会显示为不可用。
- 如果目标播放列表为空，组件会播报 `播放列表为空 : 播放列表名称`，不会开始播放。
- 如果无法识别当前播放设备，组件会播报无法确定当前播放设备。
- 如果没有听到播报，请确认屏幕阅读器或 ZDSR 相关运行环境正常。

## 重新构建

在仓库根目录运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build_output_device_switcher.ps1
```

构建完成后会重新生成：

```text
dist\foo_output_device_switcher.fb2k-component
dist\foo_output_device_switcher-x64.fb2k-component
dist\foo_output_device_switcher-fb2k-x86.fb2k-component
```
