# 播放设备切换组件使用方法

`播放设备切换` 是一个 foobar2000 组件，用于在播放菜单中快速切换 foobar2000 使用的播放设备，并通过 Tolk 播报切换后的设备名称。

## 适用环境

- Windows
- foobar2000 2.x，支持 x64 和 x86 版本
- foobar2000 1.5 / 1.6，使用 x86 兼容包

## 安装

foobar2000 2.x 可使用：

```text
foo_output_device_switcher.fb2k-component
foo_output_device_switcher-x64.fb2k-component
```

foobar2000 1.5 / 1.6 使用：

```text
foo_output_device_switcher-fb2k-x86.fb2k-component
```

foobar2000 1.x 是 32 位程序，请安装带 `fb2k-x86` 字样的安装包。后续 x86 包默认用于兼容 foobar2000 1.5 / 1.6。

在 foobar2000 中打开：

```text
文件 -> 参数选项 -> 组件
```

选择组件包安装，安装完成后重启 foobar2000。

## 菜单位置

安装后，菜单位置为：

```text
播放 -> 播放设备
```

子菜单中包含：

```text
切换到上一个播放设备
切换到下一个播放设备
```

## 使用方法

选择 `切换到下一个播放设备`，foobar2000 会切换到设备列表中的下一个播放设备。

选择 `切换到上一个播放设备`，foobar2000 会切换到设备列表中的上一个播放设备。

设备列表会循环切换。例如当前已经是最后一个设备时，继续切换下一个会回到第一个设备。

## 播报行为

切换成功后，组件会直接播报切换后的设备名称。

例如切换到耳机设备时，播报内容类似：

```text
扬声器 (Realtek(R) Audio)
```

## Tolk 文件

组件包中已经包含所需的 Tolk 文件，来源为：

```text
third_party\tolk-with-zdsr
```

安装组件包后不需要手动复制 `Tolk.dll`。

## 注意事项

- 本组件切换的是 foobar2000 的播放输出设备，不会修改 Windows 系统默认声卡。
- 如果 foobar2000 当前正在播放，切换设备可能会触发 foobar2000 重启输出流。
- 如果只有一个可用播放设备，菜单命令会显示为不可用。
- 如果没有听到播报，请确认屏幕阅读器或 ZDSR 相关运行环境正常。

## 重新构建

在仓库根目录运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build_output_device_switcher.ps1
```

构建完成后会重新生成：

```text
dist\foo_output_device_switcher.fb2k-component
dist\foo_output_device_switcher-x64.fb2k-component
dist\foo_output_device_switcher-fb2k-x86.fb2k-component
```
